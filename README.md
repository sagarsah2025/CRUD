# Book List Rest API - Node.js + Express

This is a simple REST API built using Node.js and Express that allows you to 
**manage a list of books**. Data is stored **in-memory (no database used).

### Features
- View all books (Get /books)
- View a specific book (Get /books/:id)
- Add a new book (POST /books)
- Update a book (PUT /books/:id)
- Delete a book (DELETE /books/:id)
